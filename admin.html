Configuração do GitHub (faça isso uma vez)

Para que os presentes sejam salvos automaticamente, você precisa informar os dados do seu repositório GitHub. Isso precisa ser feito apenas uma vez — os dados ficam salvos no navegador.

Como criar o token: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token. Selecione apenas a permissão repo. Cole o token abaixo.

Configuração salva!
